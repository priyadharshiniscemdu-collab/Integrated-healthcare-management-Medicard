# Integrated Healthcare Management System using MediCard

An integrated healthcare management platform focused on saving lives in emergency situations. It allows immediate access to critical patient medical histories, past medications, and allergic reactions to facilitate rapid aid.

## 🚀 Core Features
* **Emergency Medical Profile**: Immediate, secure retrieval of life-saving medical data for first responders.
* **E-Prescription Management**: Digital tracking of patient past and present medications.
* **Allergy Alerts**: Real-time checking system to prevent adverse drug reactions during immediate treatment.

## 🛠️ Tech Stack
* **Backend Framework**: Java (Spring Boot) with WebSocket support (`WebSocketConfig`).
* **Security**: Java Security Configuration (`SecurityConfig`) for role-based healthcare data protection.
* **Database**: Lightweight local structured querying capabilities.
